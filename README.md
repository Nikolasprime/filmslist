# FilmList

## Тема проекта - список фильмов. Итоговый продукт будет представлять из себя перечень фильмов, к каждому из которых будет прилагаться связанная с ним информация
Будущая система будет выглядеть следующим образом:
Пользователь - человек, желающий найти какой-либо фильм и получить о нем основную информацию.
Какие данные хранятся:
1. О фильме: описание, актерский состав, жанр, продюссеры, страна, длительность, дата выхода, возрастные ограничения
2. Об актере: имя, страна, возраст
3. О продюссере: имя, страна
Какие действия выполняются:
1. Просмотр списка всех внесенных в базу данных фильмов
2. Поиск интересующего фильма с применением фильров по определенным категориям
3. Получение подробной информации о фильме, актерском и продюссерском составе, конкретном актере
4. Возможность поставить фильму оценку "нравится" и т.д.

## Структура базы данных проекта
Основные таблицы данных:
1. film (содержит id фильма, описание, страну производства, длительность (в минутах), дату выхода, возрастные ограничения)
2. actor (содержит id актера, имя, страну, возраст)
3. genre (содержит id жанра, название)
4. producer (сожержит id продюссера, имя, страну)
Дополнительные таблицы для связи "многие к многим":
1. cast (содержит информацию об актерском составе: id фильма, id актера)
2. film_genre (содержит информацию о жанре конкретного фильма, который может относиться к нескольким жанрам сразу: id фильма, id жанра)
3. produced_by (содержит информацию о продюссерах конкретного фильма, которых может быть несколько: id фильма, id продюссера)
